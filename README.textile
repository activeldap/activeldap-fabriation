h1. ActiveLdap Fabrication

"ActiveLdap":http://ruby-activeldap.rubyforge.org/ adapter
for "Fabrication":http://fabricationgem.org/.

h2. Description

'ActiveLdap Fabrication' is an ActiveLdap adapter for
Fabrication. It means that you can use Fabrication as
fixture replacement for ActiveLdap.

For example and usage - read the
"document":http://ruby-activeldap.rubyforge.org/activeldap-fabrication/.

h2. Requirements

- "ActiveLdap":http://ruby-activeldap.rubyforge.org/
- "Fabrication":http://fabricationgem.org/
